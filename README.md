# 📚 Library Management System

A desktop application developed using **C++**, **Qt Widgets**, and **SQLite**.

## Features

- User Login
- Dashboard
- Add Book
- View Books
- Search Book
- Edit Book
- Delete Book
- Issue Book
- Return Book
- Dashboard Statistics
- Logout

## Technologies Used

- C++
- Qt 6
- SQLite
- CMake

## Database Tables

- users
- books
- issued_books
